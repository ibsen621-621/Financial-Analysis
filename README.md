# Financial-Analysis · 三表穿透分析（张新民框架）

一个可部署为 **Gemini Gem 功能**的财报分析工具：把张新民「底子—面子—日子」方法论沉淀为**可执行的指标引擎 + 风险规则 + 一页报告**。

> **资产负债表看底子 → 利润表看面子 → 现金流量表看日子 → 三表勾稽 → 风险结论**

---

## ✨ 特性

- **可复用模型**：核心口径（核心利润、核心利润率、获现率）集中管理，禁止临场心算
- **规则引擎**：7 条可执行风险规则（四高、失血增长、纸面利润、渠道压货、商誉雷、分红再融资错配、减值不足）
- **三表联动诊断**：资产/收入、收入/利润、利润/现金流勾稽
- **一页投研卡**：标准化输出战略定位、资产/资本/盈利/现金质量与最终判断
- **季度口径可比**：支持按 `period` 对"存量÷单期流量"类比率自动年化，避免季报错配
- **可部署**：FastAPI 服务 + Gemini Gem Function Calling 直连

## 🤖 部署为 Gemini Gem

1. **系统指令**：将 [`gem/gem_instructions.md`](gem/gem_instructions.md) 内容粘贴到 Gem 的 Instructions。
2. **函数声明**：在 Gem 中启用 Function Calling，导入 [`gem/function_declarations.json`](gem/function_declarations.json)。
3. **绑定后端**：将函数 `analyze_financial_statements` 指向你部署的 `POST /analyze` 服务。
4. **验证**：用 [`examples/demo_company_5y.json`](examples/demo_company_5y.json) 走通「Gem → API → 一页报告」。

> Gem 负责对话与报告叙事，API 负责固化公式与规则判断，避免模型临场改口径。

# 附：分析方法论（张新民框架）

以下为模型背后的完整分析框架，供人工复核与口径校准参考。

## 1. 使用范围与输入数据

每期分析至少准备：
- 资产负债表（期末）
- 利润表（本期）
- 现金流量表（本期）
- 报表附注（账龄、减值、商誉、债务、分红等）

建议至少覆盖近 3~5 年，便于识别趋势与异常点（可覆盖一个较完整经营周期，并区分一次性波动与结构性变化）：  
- 一般行业至少看 3 年  
- 周期性行业、重资产行业或经历并购转型的企业优先看 5 年

---

## 2. 资产负债表模块：看“底子”

### 2.1 资产端：资源配置与资产质量
1. **经营性资产 vs 投资性资产**
   - 经营性资产：货币资金、应收账款、存货、固定资产、无形资产等
   - 投资性资产：交易性金融资产、长期股权投资等
   - 关注：经营性资产占比是否长期主导（如 >70%）；投资性资产是否异常激增

2. **核心资产质量穿透**
   - 货币资金：过低有流动性风险，过高且长期闲置会拉低周转效率
   - 警惕“四高”：**高存款、高贷款、高利息、高在建工程**
   - 应收账款：看账龄结构，账龄在 2 年以上且减值不足需重点警惕（如：2年以上应收占比上升，但坏账准备覆盖率明显低于历史核销率或行业可比水平）
   - 存货：拆分原材料/在产品/库存商品，关注慢周转和减值计提不足
   - 商誉：占比过高且并购承诺未兑现，警惕未来商誉减值吞噬利润

### 2.2 负债与权益端：资本结构与含金量
3. **经营性负债（无息） vs 金融性负债（有息）**
   - 经营性负债：应付账款、应付票据、合同负债等（造血型）
   - 金融性负债：短借、长借、应付债券等（输血型）
   - 关键对比：**经营性应付款项 vs 期末存货**
     - 若经营性应付款项远大于存货：供应链话语权强
     - 若预付款项很大：对上游议价能力弱

4. **股东权益“含金量”**
   - 输血型资本：实收资本、资本公积
   - 造血型资本：盈余公积、未分配利润
   - 若未分配利润长期为负：企业内生造血能力弱，依赖外部输血

---

## 3. 利润表模块：看“面子”

### 3.1 核心利润去水分
**核心利润**  
= 营业收入 − 营业成本 − 税金及附加 − 销售费用 − 管理费用 − 研发费用 − 经营性利息费用 + 其他收益

其中“经营性利息费用”建议按以下口径界定，以保证可复用性：  
- 纳入：用于采购、生产、销售周转的借款利息（含票据贴现利息、供应链融资利息等）  
- 不纳入：用于股票/基金/理财等投资活动形成的利息或资金成本  
- 实操顺序：优先按附注披露口径划分；若未披露，再按“资金用途是否服务主营业务周转”进行归类

**核心利润率**  
= 核心利润 / 营业收入

用途：剔除资产处置、公允价值变动、补贴等非核心噪音，识别主业盈利质量。

### 3.2 结构化拆解
- 毛利率：业务护城河与定价能力
- 销售费用率：防范“高毛利 + 高销售费用”的投放依赖
- 管理费用率：治理效率和管理能力
- 研发费用：科技/制造企业未来竞争力投入强度

### 3.3 “两个脚底板”减值风险
- 信用减值损失（应收回款风险）
- 资产减值损失（存货、固定资产、商誉等）

连续大额减值通常是历史失误的集中反映，并可能被用于“财务洗澡”。

---

## 4. 现金流量表模块：看“日子”

### 4.1 核心利润获现率（利润含金量）
**核心利润获现率**  
= 经营活动现金流量 / 核心利润

- 理想状态：≥1（利润可转化为现金）
- 持续 <1：利润被应收或存货占用，存在失血风险
- 特例：直销/电商即时回款可能推高该指标，需结合渠道费用判断

### 4.2 投资活动现金流：投资效果复盘
- 扩张期投资现金流为负不一定是坏事
- 核心判断：历史投资是否带来后续营收、毛利率、核心利润改善
- 若长期无改善：未来减值风险上升

### 4.3 筹资活动现金流与分红
- 关注分红占归母净利润比例
- 警惕“一边高分红、一边高成本再融资”的股东利益错配风险

---

## 5. 三表联动诊断规则（终极勾稽）

- **资产增、收入不增**：资产配置效率低/资源闲置  
- **收入增、利润不增**：成本费用失控或价格战换规模  
- **利润增、现金流不增**：回款质量差、经营质量下滑

坚持商业常识校验：若长期出现“高现金+高息负债”等反常组合，需高度警惕财务粉饰或舞弊风险。

---

## 6. 可复用执行模板（每家企业每期一页）

1. **战略定位**：经营性资产占比 / 投资性资产占比 / 变化趋势  
2. **资产质量**：现金效率、应收账龄、存货结构、商誉占比  
3. **资本结构**：经营性负债与存货对比、有息债务压力、预付款水平  
4. **盈利质量**：核心利润、核心利润率、毛利率及三费结构  
5. **现金质量**：核心利润获现率、投资回报验证、筹资与分红匹配度  
6. **风险结论**：减值风险、流动性风险、治理与舞弊预警  
7. **最终判断**：底子（资产）-面子（利润）-日子（现金）是否一致
