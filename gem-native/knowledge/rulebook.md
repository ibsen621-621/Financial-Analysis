# 风险规则手册（rulebook.md）

> 本文件严格对应 `engine/diagnosis.py` 中的 `run_rules` 函数及 `rules/rulebook.yaml`。
> Gem 在执行规则判断时必须逐条按本文件定义的触发条件和阈值判断，不得修改。
> 规则 R1–R7，共 7 条，涵盖高风险（high）和中风险（medium）两档。

---

## 通用说明

- **判断基准年度**：所有规则默认针对**最后一个年度**（`last` = 数据中年份最大的年度）的指标值，除非明确说明为"序列判断"。
- **指标来源**：使用第二步（计算协议）中已算出的指标矩阵，不重新计算原始输入字段。
- **N/A 处理**：指标为 N/A 时，含该指标的触发条件不满足（即 N/A 视为条件不成立），规则不触发。
- **风险等级**：`high`（高）= 重大财务风险信号；`medium`（中）= 需要关注的警示信号。
- **多规则并存**：各规则独立判断，同一公司可同时触发多条规则，全部加入风险清单。

## R1 — 四高异常

| 属性 | 值 |
|---|---|
| **id** | `R1_four_high` |
| **名称** | 四高异常 |
| **等级** | high（高风险） |

### 触发条件（四项**全部**满足）

| 条件 | 指标 | 阈值 |
|---|---|---|
| 条件 1 | `last.cash_ratio`（货币资金/资产） | **≥ 0.20** |
| 条件 2 | `last.debt_ratio`（有息负债率） | **≥ 0.25** |
| 条件 3 | `last.fin_exp_ratio`（财务费用/收入） | **≥ 0.02** |
| 条件 4 | `last.cip_ratio`（在建工程/资产） | **≥ 0.10** |

> 四项必须**同时**满足才触发，任意一项不满足则不触发。

### 证据格式
```
{最新年度}: 高现金 + 高有息负债 + 高财务费用 + 高在建工程并存
```

### 建议复核项
- 核对货币资金是否受限（是否存在质押、冻结、受托管理等）
- 核对关联方资金占用

### 风险含义
高现金与高负债并存，且大量资金流入在建工程，暗示资金可能被关联方占用或存在自我循环融资；高财务费用进一步确认债务规模真实。

## R2 — 失血式增长

| 属性 | 值 |
|---|---|
| **id** | `R2_bleeding_growth` |
| **名称** | 失血式增长 |
| **等级** | high（高风险） |

### 触发条件（序列判断）

取 `cash_conversion`（核心利润获现率）序列中**最后两年**的值：
- 两个值都存在（非 N/A）
- 且两个值都 **< 0.8**

> 仅需最后两年均低于阈值，不要求更长序列。

### 阈值表

| 指标 | 阈值 |
|---|---|
| cash_conversion（最后两年均需满足） | **< 0.8** |
| 连续年数 | **2 年** |

### 证据格式
```
获现率连续 < 0.8: [倒数第二年值, 最后一年值]
```

### 建议复核项
- 应收账款增速（是否虚增收入）
- 存货增速（是否滞销积压）
- 收入确认政策（是否提前确认收入）

### 风险含义
收入与利润增长但现金不跟上，典型的"增长但失血"模式，长期不可持续，需排除收入质量问题。

## R3 — 纸面利润依赖

| 属性 | 值 |
|---|---|
| **id** | `R3_paper_profit` |
| **名称** | 纸面利润依赖 |
| **等级** | medium（中风险） |

### 触发条件（两项**同时**满足）

| 条件 | 判断方式 |
|---|---|
| 条件 1 | `core_profit_margin` 序列**首年**和**末年**均不为 N/A，且**末年值 < 首年值**（下行趋势） |
| 条件 2 | `last.non_recurring_ratio` 不为 N/A，且值 **≥ 0.30** |

### 阈值表

| 指标 | 阈值 |
|---|---|
| core_profit_margin 趋势 | 末年 < 首年（下行） |
| non_recurring_ratio（最新年度） | **≥ 0.30** |

### 证据格式
```
核心利润率下行（{首年值} → {末年值}），非经常性损益占比 {non_recurring_ratio 值}
```

### 建议复核项
- 资产处置收益（是否通过卖资产美化利润）
- 公允价值变动损益
- 政府补助（是否依赖非可持续补贴）

### 风险含义
主业盈利能力持续下滑，却依赖非经常性损益维持净利润，报告期利润"虚胖"，不可持续。

## R4 — 渠道压货嫌疑

| 属性 | 值 |
|---|---|
| **id** | `R4_channel_stuffing` |
| **名称** | 渠道压货嫌疑 |
| **等级** | medium（中风险） |

### 触发条件（三项**同时**满足）

| 条件 | 判断方式 |
|---|---|
| 条件 1 | `ar_to_rev` 序列呈**上升趋势**：去除 N/A 后，末年值 > 首年值 |
| 条件 2 | `inv_to_cogs` 序列呈**上升趋势**：去除 N/A 后，末年值 > 首年值 |
| 条件 3 | `last.cash_conversion` 不为 N/A，且值 **< 1** |

### 证据格式
```
应收/收入与存货/成本双升（ar_to_rev: {首年}→{末年}；inv_to_cogs: {首年}→{末年}）且获现率 < 1（{值}）
```

### 建议复核项
- 经销商返利政策（是否通过返利刺激经销商压货）
- 期后回款情况（报告期后应收账款实际回收率）

### 风险含义
应收和存货双双相对收入/成本上升，叠加现金获取能力不足，是典型的渠道虚增收入特征。

## R5 — 商誉减值压力

| 属性 | 值 |
|---|---|
| **id** | `R5_goodwill_risk` |
| **名称** | 商誉减值压力 |
| **等级** | high（高风险） |

### 触发条件（两项**同时**满足）

| 条件 | 指标 | 阈值 |
|---|---|---|
| 条件 1 | `last.goodwill_to_equity`（商誉/净资产） | **≥ 0.30** |
| 条件 2 | `last.goodwill_to_core_profit`（商誉/核心利润） | **≥ 3.0** |

### 证据格式
```
{最新年度}: 商誉/净资产 = {值}，商誉/核心利润 = {值}
```

### 建议复核项
- 并购标的业绩承诺完成情况
- 商誉减值测试关键假设（折现率、增长率）

### 风险含义
商誉大幅占用净资产且超过核心利润 3 倍，一旦并购标的业绩不达预期，商誉减值将大幅侵蚀净利润和净资产。

---

## R6 — 分红与再融资错配

| 属性 | 值 |
|---|---|
| **id** | `R6_capital_mismatch` |
| **名称** | 分红与再融资错配 |
| **等级** | medium（中风险） |

### 触发条件

| 指标 | 阈值 |
|---|---|
| `last.dividend_payout`（分红率） | **≥ 0.50** |

### 证据格式
```
{最新年度}: 分红率 = {值}，需核对同期再融资情况
```

### 建议复核项
- 融资成本（是否同期发债/增发，大额分红再融资本质是将股东资金转移）
- 大股东减持与资金占款

### 风险含义
大比例分红本身不一定是风险，但若同期存在大额再融资（增发/发债），实质是左手分、右手借，可能是大股东套现或隐藏资金压力的信号。

## R7 — 减值计提不足

| 属性 | 值 |
|---|---|
| **id** | `R7_provision_insufficient` |
| **名称** | 减值计提不足 |
| **等级** | high（高风险） |

### 触发条件

| 条件 | 说明 |
|---|---|
| `last.ar_provision_coverage` 不为 N/A | 即 `ar_provision` 和 `ar_aging_over_2y` 均有数据 |
| 且值 **< 0.30** | 2 年以上应收的坏账覆盖率低于 30% |

### 证据格式
```
{最新年度}: 2 年以上应收坏账覆盖率仅 {值}（ar_provision ÷ ar_aging_over_2y）
```

### 建议复核项
- 同行业可比公司坏账计提比例对比
- 是否在为未来"财务洗澡"蓄力（故意少提以后大额冲销）

### 风险含义
账龄超过 2 年的应收账款回收概率极低，坏账覆盖率不足 30% 说明计提严重不足，资产质量虚高，净利润存在高估风险。

|---|---|---|---|---|
| R1_four_high | 四高异常 | high | 四指标同时≥阈值 | 截面判断 |
| R2_bleeding_growth | 失血式增长 | high | cash_conversion 最后两年均 < 0.8 | 序列判断 |
| R3_paper_profit | 纸面利润依赖 | medium | 利润率下行 + non_recurring_ratio ≥ 0.30 | 趋势+截面 |
| R4_channel_stuffing | 渠道压货嫌疑 | medium | 双升趋势 + cash_conversion < 1 | 趋势+截面 |
| R5_goodwill_risk | 商誉减值压力 | high | goodwill_to_equity ≥ 0.30 且 goodwill_to_core_profit ≥ 3.0 | 截面判断 |

| R7_provision_insufficient | 减值计提不足 | high | ar_provision_coverage < 0.30 | 截面判断 |
